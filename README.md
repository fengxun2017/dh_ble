# dh_ble
基于nrf51的开源ble协议栈——已不再维护（没有相关硬件可以测试了）

# 目前正在 dev分支 开发基于 nordic nrf52840 的BLE 协议栈，基于最新规范新增一些必要功能，以及修改一些之前不合理的地方。实现和手机 BLE 可连接，可传输数据。
